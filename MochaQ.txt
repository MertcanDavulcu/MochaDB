No returnable commands.
{

Reset the database: RESETMOCHA
Reset the all tables: RESETTABLES
Reset the table: RESETTABLE:TableName
Create new MochaDB in directory: CREATEMOCHA:FullPath
Delete the table: DELETETABLE:TableName
Delete the sector: DELETESECTOR:SectorName
Delete the column: DELETECOLUMN:TableName:ColumnName
Delete the row: DELETEROW:TableName:RowIndex
Create new MochaDB in directory: CREATEMOCHA:DirectoryPath:Name
Rename the table: RENAMETABLE:TableName:NewName
Rename the column: RENAMECOLUMN:TableName:ColumnName:NewName
Rename the table: CREATETABLE:Name
Set the sector data: SETSECTORDATA:SectorName:NewData
Set the sector description: SETSECTORDESCRIPTION:SectorName:NewDescription
Add new sector: ADDSECTOR:Name:Data:Description
Create column in table: CREATECOLUMN:TableName:Name
Set the column description: SETCOLUMNDESCRIPTION:TableName:ColumnName:Description
Update first data in column: UPDATEFIRSTDATA:TableName:ColumnName:NewData
Update last data in column: UPDATELASTDATA:TableName:ColumnName:NewData
Update data in column by index: UPDATEDATA:TableName:ColumnName:DataIndex:NewData

}

Returnable commands.
{

Return all tables in database: GETTABLES
Return all datas in database: GETDATAS
Return table count in database: TABLECOUNT
Return first table name in database: GETFIRSTCOLUMN_NAME:TableName
Return table by name: GETTABLE:TableName
Return all column in table: GETCOLUMNS:TableName
Return all datas in table: GETDATAS:TableName
Return all rows in table: GETROWS:TableName
Return column count in table: COLUMNCOUNT:TableName
Return data count in table: DATACOUNT:TableName
Return row count in table: ROWCOUNT:TableName
Return true if table exists but return false if table not exists: EXISTSTABLE:TableName
Return true if sector exists but return false if sector not exists: EXISTSSECTOR:SectorName
Return data of sector: GETSECTORDATA:SectorName
Return description of sector: GETSECTORDESCRIPTION:SectorName
Return data count of column: DATACOUNT:TableName:ColumnName
Return row: GETCOLUMN:TableName:ColumnName
Return true if column exists in table but return false if column not exists in table: EXISTSCOLUMN:TableName:ColumnName
Return true if data exists in table but return false if data not exists in table: EXISTSDATA:TableName:ColumnName:Data
Return datas in column: GETDATAS:TableName:ColumnName
Return column description: GETCOLUMNDESCRIPTION:TableName:ColumnName

}
