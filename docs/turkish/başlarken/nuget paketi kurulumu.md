MochaDB NuGet paketini projenize ekleyin;

Açın View => Other Windows => Open Package Manager Console.

Şu kodu girin: ``Install-Package MochaDB``

İşlem bittiğinde projenize otomatik olarak eklenecek ve kullanıma hazır hale gelecektir.

> <a href="https://www.nuget.org/packages/MochaDB/">NuGet page.</a>
